# Sign-Up Form

This project is a **sign-up form page** built as part of **The Odin Project Foundations Course**. The goal of the project is to practice building and styling forms using **HTML and CSS**, while applying concepts like **form validation, pseudo-classes, and layout structuring**.

The design was provided by The Odin Project and recreated using web technologies.

---

## Project Overview

This webpage simulates a **sign-up page for an imaginary service**. It includes a visually styled form with different input fields and a sidebar containing a background image and logo.

The main objective of the project is to improve understanding of **HTML forms, CSS styling, and input validation**.

---

## Features

* Styled **sign-up form**
* Sidebar with **background image**
* Custom **logo section**
* Form fields with **basic validation**
* **Focus styling** for selected inputs
* **Invalid input styling** for password fields
* “Create Account” button
* Link for users who already have an account

## Concepts Practiced

* Structuring forms using **HTML form elements**
* Styling layouts using **CSS**
* Using **pseudo-classes** such as:

  * `:focus`
  * `:invalid`
* Working with **background images**
* Using **external fonts**
* Organizing page layout into sections

---

## Design Details

Some important design elements included in the project:

* A **semi-transparent background** behind the logo to improve readability
* A custom button color similar to the background tones (`#596D48`)
* Default input borders using a light gray color (`#E5E7EB`)
* Input fields change appearance when:

  * **Focused** (blue border with subtle shadow)
  * **Invalid** (red border)
